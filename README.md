# MetadataViewer

Программа для просмотра метаданных файлов с графическим интерфейсом (на Python + Tkinter).

![GUI](https://img.shields.io/badge/GUI-Tkinter-blue) ![Python](https://img.shields.io/badge/Python-3.8%2B-green)

## Возможности

- **Базовые метаданные файловой системы** — размер, MIME-тип, даты создания/изменения/доступа, права доступа, владелец. Доступны всегда, без сторонних библиотек.
- **Изображения** — формат, размеры, режим, DPI и полные теги **EXIF** (требуется `Pillow`).
- **Аудио** — длительность, битрейт, частота дискретизации, каналы и теги (MP3, FLAC, OGG, M4A и др.; требуется `mutagen`).
- **PDF** — количество страниц, автор, заголовок, программа-создатель и пр. (требуется `pypdf`).
- **Контрольные суммы** — MD5 и SHA-256.
- **Дерево по категориям**, сохранение отчёта в текстовый файл, перетаскивание файлов в окно (drag-and-drop при наличии `tkinterdnd2`).

## Установка

Базовая версия не требует ничего, кроме Python со стандартной библиотекой
(Tkinter входит в стандартную поставку Python на большинстве платформ).

Для поддержки изображений, аудио и PDF установите необязательные зависимости:

```bash
pip install -r requirements.txt
```

> Если при запуске возникает ошибка `No module named 'tkinter'`, установите
> системный пакет: `sudo apt install python3-tk` (Debian/Ubuntu) или
> аналогичный для вашей ОС.

## Запуск

```bash
# Открыть окно и выбрать файл через меню
python3 metadata_viewer.py

# Или сразу открыть конкретный файл
python3 metadata_viewer.py путь/к/файлу.jpg
```

## Использование

1. Нажмите **«Открыть файл…»** и выберите файл (или перетащите его в окно).
2. Метаданные отобразятся в виде дерева, сгруппированного по категориям.
3. Кнопка **«Сохранить отчёт…»** экспортирует все метаданные в текстовый файл.
4. **«Обновить»** перечитывает текущий файл.

Внизу окна показано, какие модули расширения обнаружены (`Pillow`, `mutagen`, `pypdf`).
